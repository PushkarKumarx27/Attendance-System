# 🎯 Smart Attendance System Using Facial Recognition

An intelligent and automated attendance system that uses **Computer Vision** and **Machine Learning** to detect and recognize faces in real-time, eliminating the need for manual attendance.

---

## 🚀 Features

- 📸 Real-time face detection using webcam
- 🧠 Face recognition with trained dataset
- 📝 Automatic attendance marking
- 📅 Date & time stamping
- 📊 Attendance stored in CSV/Excel format
- 🔒 Reduces proxy attendance
- ⚡ Fast and efficient processing

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Libraries & Tools:**
  - OpenCV
  - NumPy
  - Pandas
  - Flask (for web interface, if used)
- **IDE:** VS Code
